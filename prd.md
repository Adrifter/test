# 图片字幕生成器 (Subtitle Generator) 需求文档 (PRD)

## 1. 项目简介
本项目旨在开发一个网页工具，允许用户上传一张图片并为其添加多行字幕。该工具的核心特色是能够为每一行字幕生成具有相同背景的切割条带，并垂直叠加在图片底部，模拟电影或视频截图中常见的字幕叠加效果。

## 2. 功能需求

### 2.1 图片上传
- **功能描述**：用户可以通过点击“选择文件”按钮从本地选择一张图片（支持 JPG, PNG, WebP 等常见格式）。
- **交互**：选择图片后，页面应能显示该图片的预览。

### 2.2 字幕参数设置
- **字幕高度 (px)**：允许用户输入数值，控制每一行字幕背景条的高度。
- **字体大小 (px)**：控制字幕文字的大小。
- **字体颜色**：通过颜色选择器设置文字颜色（默认为白色）。
- **轮廓颜色**：通过颜色选择器设置文字轮廓/描边颜色（默认为黑色），以增强文字在复杂背景下的可读性。

### 2.3 字幕内容输入
- **多行文本框**：提供一个大型文本输入区域。
- **逻辑**：输入框中的每一行文字都将对应生成一行独立的字幕。

### 2.4 图片生成逻辑 (核心功能)
- **切割感实现**：
    - 程序需从原图底部截取一段指定高度（对应“字幕高度”）的区域作为背景模板。
    - 对于用户输入的每一行文字，程序都会复制该背景模板，并将文字居中渲染在其上。
    - 最终生成的图片由“原图”+“多行字幕条带”纵向拼接而成。
- **视觉效果**：每一行字幕的背景都是完全相同的，从而在视觉上呈现出一种规整的“切割感”。

### 2.5 操作与导出
- **生成字幕图片**：点击后触发生成逻辑，并在下方预览区域更新最终合成的图片。
- **保存图片**：允许用户将合成后的图片下载到本地。

## 3. 界面设计 (UI)
- **风格**：简洁、直观。
- **布局**：
    - 顶部：标题“图片字幕生成器”。
    - 中部：上传按钮、参数设置行、文本输入区。
    - 下部：操作按钮（生成、保存）。
    - 底部：最终结果的预览图。

## 4. 技术栈建议
- **前端**：HTML5, CSS3, JavaScript。
- **图形处理**：使用 HTML5 `<canvas>` API 进行图像裁剪、文字渲染和图片合成。

## 5. 验收标准
- 用户能够成功上传图片并看到预览。
- 输入多行文字后，点击生成按钮能看到底部叠加了对应行数的字幕条。
- 每一行字幕条的背景必须一致，且文字清晰、居中。
- 生成的图片可以正常下载并保持清晰度。
